Getting Started with HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It is used to structure content on the web.

Basic HTML Structure

An HTML document is made up of elements. Here is the basic structure of an HTML document:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Title Here</title>
</head>
<body>
    <h1>Your Main Heading</h1>
    <p>Your content goes here.</p>
</body>
</html>

Common HTML Elements

Here are some common HTML elements you will use:

  • <h1> to <h6>: Headings, where <h1> is the highest level and <h6> is the lowest.
  • <p>: Paragraphs of text.
  • <a>: Links to other pages or websites.
  • <img>: Images.
  • <ul> / <ol>: Unordered and ordered lists.

Conclusion

HTML is a fundamental building block for web development. By understanding its structure and common elements, you can start creating your own web pages!
