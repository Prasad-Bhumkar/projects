Images as Links and Image Maps

Using Images as Links

You can use images as hyperlinks by wrapping the <img> tag inside an <a> tag. Here’s an example:

<a href="https://www.example.com" title="Visit Example Website">
  <img src="https://via.placeholder.com/300" alt="Example Image" width="300" height="200">
</a>

Creating Image Maps

An image map allows you to define multiple clickable areas on a single image. You can use the <map> and <area> tags to create an image map. Here’s an example:

<img src="https://via.placeholder.com/600x400" alt="Image Map" usemap="#image-map" width="600" height="400">
<map name="image-map">
  <area shape="rect" coords="34,44,270,350" href="https://www.example.com" alt="Rectangle" title="Click to visit Rectangle Page">
  <area shape="circle" coords="337,300,44" href="https://www.example.com" alt="Circle" title="Click to visit Circle Page">
  <area shape="poly" coords="389,256,432,301,471,256,432,211" href="https://www.example.com" alt="Polygon" title="Click to visit Polygon Page">
</map>

Best Practices for Image Maps

  • Use descriptive alt text for each area to improve accessibility.
  • Use the title attribute to provide a tooltip for each area.
  • Use the href attribute to specify the link URL for each area.
  • Use the target attribute to specify the target window or tab for each area.
