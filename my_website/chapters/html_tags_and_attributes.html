HTML Tags and Attributes

What are HTML Tags?

HTML tags are the building blocks of HTML pages. They are used to define the structure and content of web pages.

Basic Tag Structure:

<tagname>content</tagname>

Common HTML Tags

Tag Description Example
<h1> to <h6> Headings <h1>Main Heading</h1>
<p> Paragraph <p>This is a paragraph</p>
<div> Division/Section <div>Container element</div>
<a> Link <a href="url">Link text</a>
<img> Image <img src="image.jpg" alt="description">

HTML Attributes

Attributes provide additional information about HTML elements and are always specified in the start tag.

Attribute Syntax:

<tag attribute="value">content</tag>

Common Attributes

Attribute Description Example
class Specifies CSS class <div class="container">
id Unique identifier <div id="header">
style Inline CSS <p style="color: blue;">
href Link reference <a href="https://example.com">
src Source file <img src="image.jpg">

Best Practices

  • Always use lowercase for tag names and attributes
  • Close all HTML elements
  • Use quotes around attribute values
  • Avoid inline styles when possible
  • Use semantic HTML elements
